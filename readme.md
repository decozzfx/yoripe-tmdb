### installation & setup your project

1. **Clone Repo** Make sure you have a **Lapakgaming-FE** clone repository.
   ```bash
   git clone https://github.com/decozzfx/yoripe-tmdb.git
   ```
2. **Install dependencies**
   ```bash
   cd yoripe-tmdb
   ```
   ```bash
   npm install or yarn
   ```
3. Setup env variable
   **Example: .env.template file to .env**
   and replace the value with yours

#### Run Application

**iOS Simulator**

```bash
react-native run-ios

npm start or yarn start
```

##### Android emulator

```bash
react-native run-android

npm start or yarn start
```
